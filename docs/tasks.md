# アプリアイコンの設定
- 実装の優先度: 1
- タスクの内容: アプリアイコン画像を作成し、`Assets.xcassets/AppIcon.appiconset/` に配置する。現在アセット構造（Contents.json）は用意済みだが、実際の画像ファイル（.png）が存在しない。macOS用の全10サイズ（16/32/128/256/512 の各1x・2x）を用意し、Info.plist の `CFBundleIconFile` も正しく設定する。
- 検討事項: アイコンデザインの方向性を決める必要がある（ユーザーに確認）。1024×1024の元画像を用意し、各サイズにリサイズするのが効率的。
