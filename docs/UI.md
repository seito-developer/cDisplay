# UI・UX設計

## メニューバーステータスアイテム

### アイコン

NSStatusItemのボタンにテンプレートイメージを設定する。テンプレートイメージを使用することで、macOSのダークモード/ライトモードに自動対応する。

2種類のアイコンを用意する：

| 状態       | スタイル               | 説明                           |
|------------|------------------------|--------------------------------|
| マスクOFF  | アウトライン/ワイヤーフレーム | 枠線のみの矩形アイコン         |
| マスクON   | ソリッド/塗りつぶし     | 塗りつぶされた矩形アイコン     |

アイコンサイズ: 18x18pt（@1x）、36x36pt（@2x）。NSImage の `isTemplate = true` を設定。

### クリック動作

アイコンをクリックするとドロップダウンメニュー（NSMenu）が表示される。

## ドロップダウンメニュー構造

```
┌─────────────────────────────────────┐
│  Toggle Mask                    ⌘M  │  ← マスクON/OFF（ショートカット付き）
│─────────────────────────────────────│
│  Aspect Ratio                    ▶  │  ← サブメニュー
│  Position                        ▶  │  ← サブメニュー
│  Click Mode                      ▶  │  ← サブメニュー
│  ☐ Show Guidelines                  │  ← チェック付きトグル
│─────────────────────────────────────│
│  Display: 2560 x 1440               │  ← マスクON時のみ表示（非活性）
│  Content: 2560 x 1440               │  ← マスクON時のみ表示（非活性）
│─────────────────────────────────────│
│  Version 1.0.0                       │  ← 非活性テキスト
│  Quit cDisplay                  ⌘Q  │
└─────────────────────────────────────┘
```

### Aspect Ratioサブメニュー

```
┌──────────────────────────┐
│  ✓ Widescreen (16:9)     │
│    Standard (4:3)        │
│    Cinemascope (2.39:1)  │
│    Square (1:1)          │
│    Vertical (9:16)       │
└──────────────────────────┘
```

- 現在選択中の項目にチェックマーク
- 選択はラジオボタン方式（排他的）
- マスク表示中に変更した場合、即座に新しい比率を適用

### Positionサブメニュー

```
┌──────────────────┐
│  ✓ Center         │
│    Top            │
│    Bottom         │
└──────────────────┘
```

- 縦方向にマスクがある場合のみ意味を持つ（横方向マスクの場合、常に水平中央）
- 現在の選択にチェックマーク

### Click Modeサブメニュー

```
┌──────────────────┐
│  ✓ Pass Through   │
│    Block          │
└──────────────────┘
```

### ディスプレイ情報

マスクON時のみ表示される2行のテキスト：

- **Display**: 現在のディスプレイ解像度（ピクセル単位）
- **Content**: マスクで区切られた表示領域のサイズ（ピクセル単位）

これらはNSMenuItemの `isEnabled = false` で非活性テキストとして表示する。

## Dockアイコン

### 表示設定

Info.plistの `LSUIElement` は設定しない（= Dockにアイコンを表示する）。

### クリック動作

`NSApplicationDelegate` の `applicationShouldHandleReopen(_:hasVisibleWindows:)` をオーバーライドし、マスクのON/OFFをトグルする。

```swift
func applicationShouldHandleReopen(
    _ sender: NSApplication,
    hasVisibleWindows flag: Bool
) -> Bool {
    maskService.toggle()
    return false
}
```

`return false` とすることで、デフォルトの「ウィンドウを表示する」動作を抑制する。

### 右クリックメニュー

Dockの右クリックメニューはmacOS標準の動作に委ねる（カスタマイズしない）。

## オンボーディング

### 表示条件

- 初回起動時のみ表示
- `UserDefaults` の `hasCompletedOnboarding` フラグで管理
- 完了後は再表示しない

### フロー

3ステップのウォークスルー形式：

**ステップ1: マスクの切り替え**
- メニューバーアイコンとDockアイコンによるON/OFF操作を説明
- キーボードショートカットの存在を紹介

**ステップ2: アスペクト比と設定**
- メニューバードロップダウンからの設定変更方法を説明
- アスペクト比プリセットの一覧を表示

**ステップ3: 画面キャプチャとの併用**
- OBS等の画面キャプチャツールで「画面全体」をキャプチャすると、マスク込みで取り込まれることを説明
- これが主要な利用方法であることを案内

### ウィンドウ仕様

- 中央配置のモーダルウィンドウ
- 「Next」「Back」「Done」ボタンで操作
- 各ステップにイラストまたはスクリーンショットを表示（v1ではテキストのみでも可）

## キーボードショートカット

### デフォルト設定

マスクON/OFFトグル用のグローバルショートカットを提供する。

キーの選定基準：
- OBS（Ctrl+各種）との競合を避ける
- macOS標準ショートカットとの競合を避ける
- 覚えやすい組み合わせ

候補: `⌘⇧M`（Command + Shift + M）— "M"はMaskの頭文字。

### ユーザーカスタマイズ

メニューバードロップダウンの「Toggle Mask」項目にショートカットが表示される。変更方法はmacOSのシステム環境設定 > キーボード > ショートカットに委ねるか、アプリ内で録音（Record Shortcut）UIを提供するか、実装時に決定する。
