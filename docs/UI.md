# UI・UX設計

## メニューバーステータスアイテム

### アイコン

SF Symbolsを使用し、テンプレートイメージとして設定する。macOSのダークモード/ライトモードに自動対応する。

| 状態 | SF Symbol | 説明 |
|------|-----------|------|
| OFF | `rectangle.dashed` | 枠線のみの矩形アイコン |
| ON | `rectangle.fill` | 塗りつぶされた矩形アイコン |

### クリック動作

アイコンをクリックするとドロップダウンメニュー（NSMenu）が表示される。

## ドロップダウンメニュー構造

```
┌─────────────────────────────────────┐
│  Enable                             │  ← 表示ON/OFF（状態に応じて Enable/Disable）
│─────────────────────────────────────│
│  Resolution                      ▶  │  ← サブメニュー（ターゲット解像度一覧）
│─────────────────────────────────────│
│  Native: 2560 × 1440               │  ← ネイティブ解像度（非活性テキスト）
│  Active: Resolution (1920 × 1080)   │  ← ON時のみ表示、表示方式の情報
│─────────────────────────────────────│
│  Version 1.0.0                      │  ← 非活性テキスト
│  Quit cDisplay                      │
└─────────────────────────────────────┘
```

### Resolutionサブメニュー

ターゲット解像度をアスペクト比ごとにグループ化して表示する：

```
┌────────────────────────────┐
│  — 16:9 —                  │  ← セクションヘッダー（非活性）
│  ✓ 1920 × 1080             │
│    1600 × 900              │
│    1280 × 720              │
│─────────────────────────── │
│  — 4:3 —                   │
│    1600 × 1200             │
│    1280 × 960              │
│    1024 × 768              │
│─────────────────────────── │
│  — 2.39:1 —                │
│    1920 × 803              │
│─────────────────────────── │
│  — 1:1 —                   │
│    1080 × 1080             │
│    720 × 720               │
│─────────────────────────── │
│  — 9:16 —                  │
│    1080 × 1920             │
│    720 × 1280              │
└────────────────────────────┘
```

- 現在選択中の項目にチェックマーク
- 選択はラジオボタン方式（排他的）
- 選択時に即座にターゲット解像度を適用

### ディスプレイ情報

メニューに表示される情報行：

- **Native**: 現在のネイティブ解像度（常時表示、非活性テキスト）
- **Active**: ON時のみ表示。適用中の表示方式と解像度を表示（非活性テキスト）

## Dockアイコン

### 表示設定

`NSApp.setActivationPolicy(.regular)` でDockにアイコンを表示する。

### クリック動作

`NSApplicationDelegate` の `applicationShouldHandleReopen(_:hasVisibleWindows:)` で表示のON/OFFをトグルする。

```swift
func applicationShouldHandleReopen(
    _ sender: NSApplication,
    hasVisibleWindows flag: Bool
) -> Bool {
    statusBarController?.toggle()
    return false
}
```

`return false` とすることで、デフォルトの「ウィンドウを表示する」動作を抑制する。

### 右クリックメニュー

Dockの右クリックメニューはmacOS標準の動作に委ねる（カスタマイズしない）。
